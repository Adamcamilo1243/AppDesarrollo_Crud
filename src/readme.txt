

Proyecto CRUD ---- Gestion de productos-online

Proyecto desplegado en Firebase

Firebase es una plataforma de desarrollo móvil y web de Google que proporciona una variedad de herramientas y servicios para construir
aplicaciones de alta calidad. Firebase está diseñado para simplificar el proceso de creación de aplicaciones y mejorar la eficiencia del desarrollo, al
proporcionar una infraestructura escalable y flexible para las aplicaciones.


Descripción del proyecto.


Es un desarrollo tipo CRUD, el cual consume informacion de la siguiente API RES: https://fakestoreapi.com/
*Se uso la libreria de bootstrap
*Se uso DataTable, Biblioteca JavaScript de código abierto que se utiliza para mejorar la interacción y visualización de datos en una tabla HTML.
*en la tabal se muestran los datos de la API RES consumida.
*la tabla contiene 3 opciones de botones Crear, Editar,eliminar.
